# Todo

Take notes on the commandline.
As of this time only linux based platforms are supported.
Feel free to contribute to support other platforms.

## Features
- Add new note 
- Update an existing note 
- Delete a note 
- List all notes 

## Installation
Clone the github repository
Build the program.
```bash
bash build.sh
```

## Usage
```bash
todo help
```
